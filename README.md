# 🎟️ Microsserviço de Vendas de Ingressos — Atividade Prática 01

Este repositório contém a **Atividade Prática 01** do projeto de Gerenciamento de Tickets, com foco na **implementação do microsserviço de Vendas (Sales)**.

## 📌 Descrição

A proposta desta atividade prática é o desenvolvimento de um microsserviço responsável por **registrar eventos disponíveis** e **gerenciar a venda de ingressos** para os usuários. O serviço faz parte de uma arquitetura baseada em microsserviços para o sistema de gerenciamento de tickets.
